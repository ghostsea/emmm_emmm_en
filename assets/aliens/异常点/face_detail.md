外星人核心机制：

当外星人揭示时，若地球所在的扇区坐标系的x坐标为m，则在[m,4]、[m-3,4], [m+3,4]扇区各放置一个异常标记（assets\aliens\异常点\?_?），这3个标记均被称为 异常 。标记放到该扇区边缘位置，不要放到中间。异常标记选择a\b\c各一个，每一块随机选择1或2，如放置b_1、a_2、c_2到3个位置。

这几个标记不会随旋转移动，且要记录这几个坐标和对应的标记，当地球再次对准相应扇区时（如x=m、m+3、m-3）时，会触发对应的奖励，在外星人面板上痕迹更靠上的位置（顺位为5的最靠下，顺位为1的最靠上，且1上面后放的更靠上）的玩家，会获得相关奖励。如果外星人面板上没有标记，则无人获得相关奖励。奖励和对应关系如下：

| 编号 | 奖励    | 对应痕迹颜色 |
| ---- | ------- | ------------ |
| a_1  | 1信用点 | 粉色         |
| a_2  | 4分     | 粉色         |
| b_1  | 2宣传   | 黄色         |
| b_2  | 1精选   | 黄色         |
| c_1  | 1数据   | 蓝色         |
| c_2  | 1能量   | 蓝色         |

触发异常即地球的扇区坐标中x值=m、m+3、m-3。下一个即将触发的异常，即旋转中（x值减小），下一个等于的[m、m+3、m-3]中对应的那个扇区。

如x=4时触发外星人揭示，则m为4、7、1。x扇区坐标为4、7、1的扇区为异常扇区。此时下一个要触发的异常为1扇区（地球逆时针旋转）。
