# 奥陌陌实现说明

机制来源：`face_detail.md`。通用外星人生命周期见 `docs/alien-design.md`。

## 当前状态

- 代码入口尚未迁移为专属模块。
- 设计源文档已存在：`assets/aliens/奥陌陌/face_detail.md`。
- 后续实现时应按 `docs/alien-design.md` 的生命周期和检查清单接入状态、坐标、撤销与日志。

## 设计要点

- 奥陌陌有专属资源“化石”，图标为 `assets/aliens/奥陌陌/fossil.webp`。
- 揭示后需要替换 `wheel3.png` 为 `wheel3_amm.png`，保持层级、角度和尺寸。
- 新增的奥陌陌星球与土星、火星同一圈，位置为土星 `x-2`、木星 `x+2`。
- 奥陌陌所在扇区既是星球，也是可扫描扇区。
- 奥陌陌扇区只有 3 个数据：1 号位扫描得 1 分，3 号位扫描得 2 分。
- 完成奥陌陌扇区时，每个参与扫描的玩家获得 1 化石。
- 扫描奥陌陌时与当前该扇区所在位置对应的外圈星云共享二选一扫描机制。
- 在奥陌陌星球的火箭可以执行环绕、登陆；环绕/登陆标记放到外星人面板上方对应区域。
- 正面痕迹为粉/黄/蓝各 1-5 号位，1 号位可无限向上追加，其它位置单占用。
- 痕迹奖励和奥陌陌牌模型详见 `face_detail.md`。

## 实现注意事项

- 奥陌陌会改变太阳系版图和扫描目标集合，实现时要把星球、扇区扫描、扇区结算、环绕/登陆、旋转推动全部纳入统一状态。
- `wheel3_amm.png` 切换和新增星球位置属于揭示副作用，揭示初始化应写不可逆屏障。
- 奥陌陌化石是专属资源，应明确挂在玩家状态或 `alienGameState.amm`，不要只存在 UI。
- 外星人牌涉及扫描奥陌陌、登陆奥陌陌、移除化石等效果，需要复用主行动效果队列和任务系统。
