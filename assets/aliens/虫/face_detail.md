外星人核心机制：

外星人揭示后，会在随机把7个化石标记(fossil_0*.png)，其中3个放置在土星，3个放置在木星（放在wheel3上面），每个星球上只展示一个标记背景（fossil_back.webp），特定情况下，玩家可以获得权限点击该图标然后可以弹窗显示具体的化石正面内容【需要开发该查看机制】。第7个化石标记放在虫族面板默认的2个蓝色圆圈上方的有标记的那个蓝色圆圈内容，使用正面。

7个化石标记的奖励分别为：

| 编号      | 奖励     |
| --------- | -------- |
| fossil_01 | 3宣传    |
| fossil_02 | 7分      |
| fossil_03 | 3分1精选 |
| fossil_04 | 2盲抽    |
| fossil_05 | 2数据    |
| fossil_06 | 2能量    |
| fossil_07 | 2信用点  |

放置在外星人面板上的化石标记，正面朝上，且可以通过获得蓝色外星人痕迹放置标记以获得奖励。

虫族的卡牌可以实现查看木星或土星上的化石标记，然后将化石标记运到特定区域或者获得特定化石标记的奖励。

如果化石标记运回特定地点完成虫族专属卡牌任务，则可以把化石标记放到虫族面板一个空的蓝色圆圈内，从而变成可以放置token的位置。

运输化石：当玩家通过虫族的卡牌，标记一个化石并拾起时，该化石（还是以back为图标，但是该玩家能够鼠标放置到化石上再次查看对应的正面）上放置一个玩家token，并按照火箭的可用位置放置该（化石+token的组合图标），此时该化石为被运输，并且视为“火箭”，玩家可以通过移动来控制该化石，并且该化石进入行星、彗星、橙2科技下小行星也会获得宣传。能移动火箭的卡牌也能移动化石，化石也可以点击触发快速移动效果。一旦化石到达其目的地，则立刻从太阳系版图删除并结算相关卡牌任务的效果。
